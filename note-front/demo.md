# 智能笔记本应用演示

## 🎯 项目概述

这是一个功能完整的Vue 3笔记本应用，具有以下特点：

### 🌟 界面设计
- **现代化UI设计**：使用Element Plus组件库，界面美观大方
- **三栏布局**：左侧笔记本列表，中间笔记列表，右侧编辑器
- **响应式设计**：适配不同屏幕尺寸
- **渐变背景**：登录注册页面采用美观的渐变背景

### 🚀 核心功能

#### 1. 用户认证系统
- **注册功能**：支持手机号注册，密码确认验证
- **登录功能**：手机号登录，验证码验证
- **安全性**：JWT Token认证，自动登录状态维护

#### 2. 笔记本管理
- **创建笔记本**：支持添加名称和描述
- **笔记本列表**：显示所有笔记本及笔记数量
- **删除笔记本**：支持删除不需要的笔记本

#### 3. 笔记管理
- **创建笔记**：Markdown格式支持
- **编辑笔记**：实时编辑，自动保存
- **搜索功能**：按标题和内容搜索笔记
- **置顶功能**：重要笔记可置顶显示
- **标签管理**：为笔记添加标签分类

#### 4. 智能功能
- **自动保存**：防止数据丢失
- **字数统计**：实时显示笔记字数
- **阅读统计**：记录笔记阅读次数
- **时间筛选**：可查看最近7天的笔记

## 🛠️ 技术实现

### 架构设计
- **Vue 3 Composition API**：现代化的组件开发方式
- **Pinia状态管理**：轻量级、类型安全的状态管理
- **Vue Router 4**：声明式路由管理
- **Element Plus**：企业级UI组件库
- **Axios**：HTTP客户端，支持请求拦截和响应处理

### 项目结构
