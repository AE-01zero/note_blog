# 智能笔记本应用

一个基于Vue 3 + Element Plus的现代化笔记本应用，支持用户注册登录、笔记本管理、笔记编辑等功能。

## 功能特性

- ✅ 用户注册登录（带验证码）
- ✅ 笔记本管理（创建、编辑、删除）
- ✅ 笔记管理（创建、编辑、删除、搜索）
- ✅ 笔记置顶功能
- ✅ 标签管理
- ✅ 自动保存
- ✅ 响应式设计
- ✅ 美观的现代化UI

## 技术栈

- Vue 3 (Composition API)
- Element Plus UI组件库
- Vue Router 4
- Pinia 状态管理
- Axios HTTP客户端
- Vite 构建工具

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

## 项目结构

```
src/
├── api/           # API接口
├── components/    # 公共组件
├── router/        # 路由配置
├── store/         # 状态管理
├── views/         # 页面组件
├── App.vue        # 根组件
└── main.js        # 应用入口
```

## 接口说明

后端接口基于提供的API文档实现，包括：

- 用户认证（注册、登录、获取用户信息）
- 验证码获取
- 笔记本管理（CRUD操作）
- 笔记管理（CRUD操作、搜索、置顶）
- 文件上传
- 标签管理

## 页面功能

### 登录页面
- 手机号码登录
- 验证码验证
- 美观的渐变背景设计

### 注册页面
- 用户注册功能
- 密码确认验证
- 验证码验证

### 笔记本主页
- 三栏布局设计
- 左侧：笔记本列表和过滤器
- 中间：笔记列表
- 右侧：笔记编辑器

### 主要功能
- 笔记本分类管理
- 笔记搜索功能
- 笔记置顶功能
- 标签管理
- 自动保存
- 字数统计

## 使用说明

1. 注册新用户或使用现有账号登录
2. 创建笔记本进行分类管理
3. 在笔记本中创建和编辑笔记
4. 使用搜索功能快速找到笔记
5. 为笔记添加标签进行更好的组织
6. 使用置顶功能标记重要笔记

## 注意事项

- 确保后端API服务运行在 `http://localhost:8888`
- 项目使用了Element Plus组件库，UI风格现代化
- 支持自动保存，防止数据丢失
- 响应式设计，适配不同屏幕尺寸
